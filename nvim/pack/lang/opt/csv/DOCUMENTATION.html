1 Introduction

This is a ultra-minimal plug-in for CSV viewing, not editing. This plug-in Features include

  • highlight columns via number or placing the cursor on top of it.
  • move cursor to the highlighted column.
  • easy navigating from one column to the next/previous.
  • select the entry under the cursor in the column.
  • display the column names in a split window for easy navigation.
  • search for pattern in the highlighted column.
  • conceal the deliminator for easy viewing.

Although editing the CSV file is not the aim, the following features are included in this plug-in:

  • convert csv into tsv (safely) and vice versa.

still some functions in this plug-in that can modify the CSV file, but modifying the file is not the main focus of plug-in.

  • simple: a file is said to be simple if there is no \\" in the document, i.e., there are no quotes immediately after an escaped backslash. A simple file can contain quotes, escaped quotes or deliminator within quotes.
  • super_simple: a file is said to be super_simple if there is no quotes, escaped quotes or escaped deliminator in the file.
  • dangerous: a file is said to be dangerous if there are escaped Backslashs in the file. A dangerous file is not necessary not simple, but a not simple file is dangerous.
